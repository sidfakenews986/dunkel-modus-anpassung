# Dunkel Modus Anpassung

## Beschreibung
Diese Anwendung ermöglicht es Benutzern, ihre eigenen Dunkelmodus-Themen für verschiedene Webseiten und Anwendungen zu erstellen und anzupassen. Ziel ist es, die Sichtbarkeit und den Komfort für Benutzer zu verbessern, die in dunklen Umgebungen arbeiten.

## Features
- Anpassung von Farben, Schriftarten und Hintergründen für den Dunkelmodus
- Vorschau-Funktion zur Anzeige von Änderungen in Echtzeit
- Möglichkeit, benutzerdefinierte Themen zu speichern und zu teilen
- Unterstützung für verschiedene Browser
- Einfache Implementierung eines Dunkelmodus in bestehenden Projekten

## Installation
1. Klone das Repository: `git clone https://github.com/benutzername/dunkel-modus-anpassung.git`
2. Navigiere in das Verzeichnis: `cd dunkel-modus-anpassung`
3. Installiere die Abhängigkeiten: `npm install`
4. Starte die Anwendung: `npm start`
